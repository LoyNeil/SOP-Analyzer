ProcessLens

Turn Zoom meeting transcripts and SOP documents into structured process assets in seconds.

ProcessLens analyses meeting recordings and existing SOPs to automatically generate Standard Operating Procedures, interactive swimlane diagrams, and bottleneck reports — all from a single file upload.

✨ Features

Transcript → SOP — Upload a Zoom .vtt or .txt transcript and get a fully structured SOP with roles, steps, happy/unhappy paths, handoffs, and decision points
Transcript → Process Map — Auto-generated interactive swimlane diagram with draggable nodes, editable labels, and PDF export
Transcript → Bottleneck Analysis — Detects 9 signal types (manual effort, missing SLAs, handoff delays, tool friction, and more) with severity scoring and actionable suggestions
SOP Document → Restructure — Upload an existing .docx or .pdf SOP and map it faithfully into the structured schema — no paraphrasing, no invented content
SOP Document → Process Map — Derive a swimlane diagram directly from your existing SOP roles and steps
SOP Document → Bottleneck Analysis — Structural gap detection on written SOPs (missing owners, absent validation gates, ambiguous steps)
Inline editing — Every field in the generated SOP is editable directly in the UI
Export to Word — Download the structured SOP as a formatted .docx file
Copy as rich HTML — Paste directly into Confluence, Notion, or Word with formatting preserved
Fullscreen diagram editor — Open the swimlane diagram in a dedicated tab with undo/redo, bend-able arrows, and multi-lane support
